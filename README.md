# Resolve Aqui

O objetivo do sistema é criar um ambiente digital de social-commerce para serviços, promovendo a visibilidade de profissionais autônomos, facilitando a tomada de decisão dos clientes através de portfólios e garantindo segurança nas transações por meio de um sistema de pagamento intermediado
